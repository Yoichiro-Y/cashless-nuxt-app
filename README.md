- キャッシュレス決済のレビューサイトです
- フロントエンドはnuxt.js、バックエンドはfirebaseを用いました。また、TailwindCSSを使用しました。
- ユーザー登録・ログイン・ログアウト機能（Google、メールアドレス）、レビューの投稿機能、レビューの削除・編集機能（paymentのレビューのみ）、決済方法新規登録機能、キャンペーン新規登録機能
